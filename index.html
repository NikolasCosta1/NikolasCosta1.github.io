Este é um projeto para demonstrar o javascript.

A página lista deve mostrar uma tabela com lista contendo 4 colunas:

Nome, Email, DataNascimento, Botão Deletar

A página api, deve fazer uma chamada para a API “https://jsonplaceholder.typicode.com/posts/1” e mostrar o resultado na tela.

A página botão, deve conter 3 botões “Botão 1”, “Botão 2” “Botão 3” e 3 divs com texto “Div 1”, “Div 2”, “Div 3”.

As divs devem começar escondidas(“hidden”). Ao clicar em um botão a div correspondente deve ser exibida e esconder as outras.
